An App To Learn Technologies To Add to Your Resume

We all want to be able to add a long list of technologies to our resume, but I wanted a little app to track all these technologies and how deep my understanding really is. This app captures my favorite timeline for studying a computer science technology. To use it, type a technology, like "Django" into the text box and click Add Technology. To save your list, click download JSON. To use an already created list, click "Choose File". I use the Next Step field in each technology as a bookmark to track the youtube video, or documentation page I was last watching or reading.
